# Python Text Town Life Game

A text-based town life game where the player works to save enough money to buy a house while managing daily expenses, energy, and unforeseen events.

## Features

* Multiple jobs with varying risk and reward
* Random events
* Fluctuating food prices
* Resource management (energy and money)
* Sickness mechanic
* Save and load functionality

## Concepts Implemented

* Object-Oriented Programming (OOP)
* Functions
* Random module
* File I/O
* Game loop

## Requirements

* Python 3.x

## How to Run

1. Run the main game file.
2. Start a new game or load an existing save.
3. Earn money by working different jobs while managing your energy and daily expenses.
4. Overcome random events and save enough money to purchase a house.
5. Save your progress and continue later if desired.

## Future Improvements

* Additional jobs with unique mechanics
* More random events and encounters
* Inventory system
* Expanded progression and multiple endings
* Difficulty settings

## Goal

The objective of the game is to balance income, expenses, and energy while adapting to unpredictable events. Careful resource management is essential to eventually earn enough money to buy a house and complete the game.
