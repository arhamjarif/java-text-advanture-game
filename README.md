# Python Text Town Life Game

A text-based town life game where the player works to save enough money to buy a house while managing daily expenses, energy, and unforeseen events.

## Features

* Multiple jobs with varying risk and reward
* Random events
* Fluctuating food prices
* Resource management (energy and money)
* Sickness mechanic
* Save and load functionality

## Concepts Implemented

* Object-oriented programming (OOP)
* Functions
* Random module
* File I/O
* Game loop
